### üè´ **Scalable Enterprise School ERP System**
*   **Challenge:** Replace disparate legacy systems with a unified, modern, cloud-native platform for improved efficiency and data consistency across a large educational institution.
*   **Solution:** Architected and led the development of a microservices-based ERP using Node.js (TypeScript), React, and MongoDB. Implemented robust role-based access control, real-time event notifications (WebSockets/SNS), automated reporting, and integrated third-party payment gateways.
*   **Architecture & Deployment:** Containerized services with Docker, orchestrated on AWS EKS with Helm charts. Leveraged AWS RDS (PostgreSQL) for relational data, DynamoDB for high-throughput metadata, S3 for asset storage, and ElastiCache (Redis) for session management/caching. Established a full CI/CD pipeline using GitHub Actions & Argo CD for GitOps deployment to EKS. Implemented comprehensive monitoring with Prometheus, Grafana, and CloudWatch, integrating PagerDuty for critical alerting.
*   **Impact:** Reduced manual data entry by ~60%, improved reporting accuracy, enhanced user satisfaction for admin, staff, and students. Achieved high availability and scalability to handle peak loads during admission/result periods.

### ü§ñ **AI-Enhanced Customer Interaction Platform**
*   **Challenge:** Reduce support agent workload for common queries and provide instant 24/7 assistance on the company's primary web application.
*   **Solution:** Developed an AI chatbot integrating Google Dialogflow for NLU, with a Node.js backend service (webhook) hosted on AWS Lambda for dynamic data retrieval and complex logic fulfillment. Built a seamless React component for frontend integration with graceful fallback mechanisms.
*   **Architecture & Deployment:** Serverless architecture leveraging API Gateway, Lambda, and DynamoDB for conversation state/context. CI/CD managed via Serverless Framework and AWS CodePipeline. Trained and iterated on Dialogflow intents/entities for improved accuracy. Monitored Lambda performance and errors using CloudWatch Logs & X-Ray.
*   **Impact:** Deflected ~40% of Tier-1 support queries, improved customer engagement metrics, and provided faster resolution times.

### üèòÔ∏è **High-Performance Real Estate Listing Aggregator**

*   **Solution:** Developed a MERN stack application with optimized database indexing (MongoDB geospatial indexes) and efficient backend API design. Implemented server-side rendering (SSR) with Next.js for improved SEO and initial load performance. Integrated Mapbox API for interactive map displays and geocoding. Utilized AWS S3 with CloudFront CDN for optimized image delivery.
*   **Architecture & Deployment:** Deployed on AWS EC2 Auto Scaling Group behind an Application Load Balancer. Used MongoDB Atlas as the managed database service. Implemented background job processing (e.g., image resizing) using SQS and Lambda. Set up CI/CD with Jenkins deploying to EC2 instances.
